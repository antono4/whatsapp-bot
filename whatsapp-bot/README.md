# WhatsApp Bot

Bot WhatsApp sederhana dengan fitur auto-reply.

## Fitur
- Auto-reply berdasarkan keyword
- Respon custom untuk pesan masuk
- Logging pesan

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Jalankan bot:
```bash
python main.py
```

3. Scan QR Code dengan WhatsApp Anda
